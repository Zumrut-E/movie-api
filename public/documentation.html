Movie API Documentation

Endpoints

Request URL Response
GET /movies Returns a list of all movies
GET /movies/:title Returns data about a single movie by title
GET /genres/:name Returns data about a genre by name
GET /directors/:name Returns data about a director by name
POST /users Allows new users to register
PUT /users/:username Allows users to update their user info
POST /users/:username/favorites/:movieId Allows users to add a movie to their list of favorites
DELETE /users/:username/favorites/:movieId Allows users to remove a movie from their list of favorites
DELETE /users/:username Allows existing users to deregister

Endpoint Details

GET /movies

Returns a list of all movies.

URL: /movies

Method Type: GET

Response Format: JSON

Example Response:

    [
        {
            "title": "Inception",
            "description": "A mind-bending thriller",
            "genre": "Sci-Fi",
            "director": "Christopher Nolan",
            "imageUrl": "inception.jpg",
            "featured": true
        },
        {
            "title": "The Matrix",
            "description": "A computer hacker learns about the true nature of his reality",
            "genre": "Sci-Fi",
            "director": "Lana Wachowski",
            "imageUrl": "matrix.jpg",
            "featured": false
        }
    ]

GET /movies/:title

Returns data about a single movie by title.

URL: /movies/:title

Method Type: GET

Response Format: JSON

Example Response:

    {
        "title": "Inception",
        "description": "A mind-bending thriller",
        "genre": "Sci-Fi",
        "director": "Christopher Nolan",
        "imageUrl": "inception.jpg",
        "featured": true
    }

GET /genres/:name

Returns data about a genre by name.

URL: /genres/:name

Method Type: GET

Response Format: JSON

Example Response:

    {
        "name": "Sci-Fi",
        "description": "Science fiction movies"
    }

GET /directors/:name

Returns data about a director by name.

URL: /directors/:name

Method Type: GET

Response Format: JSON

Example Response:

    {
        "name": "Christopher Nolan",
        "bio": "British-American film director",
        "birthYear": 1970,
        "deathYear": null
    }

POST /users

Allows new users to register.

URL: /users

Method Type: POST

Request Body:

    {
        "username": "newUser"
    }

Response Format: Text

Example Response: User registered successfully

PUT /users/:username

Allows users to update their user info.

URL: /users/:username

Method Type: PUT

Request Body:

    {
        "username": "updatedUsername"
    }

Response Format: Text

Example Response: User info updated successfully

POST /users/:username/favorites/:movieId

Allows users to add a movie to their list of favorites.

URL: /users/:username/favorites/:movieId

Method Type: POST

Response Format: Text

Example Response: Movie with ID [movieId] added to [username]'s favorites

DELETE /users/:username/favorites/:movieId

Allows users to remove a movie from their list of favorites.

URL: /users/:username/favorites/:movieId

Method Type: DELETE

Response Format: Text

Example Response: Movie with ID [movieId] removed from [username]'s favorites

DELETE /users/:username

Allows existing users to deregister.

URL: /users/:username

Method Type: DELETE

Response Format: Text

Example Response: User [username] deregistered successfully
